## Install Dependencies
### Ubuntu
```
sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev libenet-dev libopenal-dev libalut-dev
```
### Arch Linux
```
sudo pacman -S sdl2 sdl2_image sdl2_ttf enet openal freealut
```

## Build and Run
```
make
./myapp
```
